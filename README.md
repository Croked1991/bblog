# Just a blog

There is a possibility to:
  - add new post
  - delete old post 
  - search the posts by title
  - sort posts by title or by body
 
In the app uses (or will be uses at soon):
  - react js
  - redux
  - react-redux
  - redux thunk
  - JSON Placeholder API
  - axios
  - modal windows
  - authorisation form
  - custom UI components
  - and other
  
 Welcome to try it)))
